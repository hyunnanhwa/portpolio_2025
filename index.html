안녕하세요. 프론트엔드 개발자를 지향하는 퍼블리셔 현난화 입니다.

아래는 제가 몸 담았던 회사에서의 경험과 주요 업무를 정리해두었습니다. 좌우 스와이프를 통해 각 이력을 확인해주세요!

주식회사 써니

|

2024.10 - 2025.01

자동차 용품 전문 인하우스 회사에서 웹 퍼블리셔로 근무하며,
자사 콘텐츠 중심의 웹 구축 및 운영 업무를 담당했습니다.

제품 촬영, 상세페이지 제작 등 이커머스 전반의 콘텐츠 제작을 경험했으며,
카페24와 스마트스토어 기반으로 기업형 페이지와 자사몰을 구축하고,
쇼핑라이브를 지원하며 라이브 커머스 운영에 대한 경험을 쌓았습니다.

또한 회원 카페 내 이벤트를 위한 간단한 게임형 웹 코드 (제비뽑기, 사다리타기 등)를
직접 제작 및 배포하며, 사용자 참여를 유도하는 인터랙티브 요소도 구현했습니다.

애드펀(디플래닛)

|

2024.04 - 2024.10 / 2022.02 - 2023.05

웹 에이전시로, 주로 병원 및 선사 관련 업무를 진행했습니다.
퍼블리셔로 입사 하였으나, 실무에서 역할 구분 없이 다양한 영역을 경험하게 되었고,
그 과정에서 프론트엔드 개발과 데이터베이스 설계를 접하게 되며
자연스레 관심을 갖게 되었습니다.

주요 업무로는 UI, UX 개선 및 유지보수, API 연동 작업을 비롯하여
환자별 맞춤 예약 페이지, 선사용 관리 페이지 개발 등이 있으며,
다양한 업체와 협업 속에서 실무 역량을 넓히는 계기가 되었습니다.

선영디자인

|

2023.09 - 2023.12

웹 에이전시로, 카페24 기반의 기업 홈페이지 및 쇼핑몰 구축 업무를 담당했습니다.
HTML, CSS, JavaScript 를 중심으로 다양한 기능형 UI, UX 를 구현하며,
웹 페이지 구조에 대한 이해도를 더 높일 수 있었습니다.

짧은 기간이었지만 디자이너와의 협업을 통해 커뮤니케이션의 중요성을 체감했고,
카페24 플랫폼의 운영 방식과 정책에 대한 이해를 쌓는 계기가 되었습니다.

저에 대해 궁금하실 만한 내용을 담아봤어요 🙋🏻‍♀️

Q. 프론트엔드를 지향하는 이유는?

이전 회사에서 프론트엔드 업무를 처음 접했고, 어렵지만 문제를 해결해 나가는 과정이 즐거웠습니다.
퍼블리셔보다 사용자와의 상호작용이 더 많은 작업이 매력적으로 느껴졌고,
기초 지식을 쌓으며 React와 같은 프레임워크를 활용해 꾸준히 공부 중입니다.

Q. 일에 있어 가장 중요하게 생각하는 가치는?

작은 작업이라도 누군가에게 도움이 되는 보람을 중요하게 생각합니다.
사용자와의 상호작용에서 느껴지는 실질적인 반응도 좋지만,
회사 동료들과의 협업 과정에서 서로에게 기여하며 함께 완성해가는 경험을 큰 보람으로 느낍니다.

Q. 앞으로 어떤 개발자가 되고 싶은지?

변화에 유연하게 대응하고 지속적으로 성장하는 개발자가 되고 싶습니다.
온라인 강의, 관련 서적, 사이드 프로젝트 등을 통해 기초와 실무 감각을 함께 키워가며,
실무 역량을 갖춘 개발자로 차근차근 나아가고 싶습니다.

이런 기술들을 사용 했어요.

프로젝트를 진행하며 실제로 활용한 기술과 툴들을 정리해두었습니다.

프론트엔드

  • HTML

    웹 페이지의 구조를 마크업하는 데 사용했습니다.

  • CSS

    반응형 웹 및 스타일 커스터마이징에 사용했습니다.

  • Java Script

    동적 기능 구현에 사용했습니다.

  • API

    서버에서 데이터를 받아와 동적으로 화면에 표시했습니다.

백엔드

  • PHP

    서버에서 동적으로 웹 페이지를 생성하고 폼 데이터 처리에 사용했습니다.

  • Node.js

    프론트엔드에서 사용한 데이터를 처리하는 서버 개발에 사용했습니다.

  • MsSQL

    DB 테이블 설계 및 사용자 관리에 사용했습니다.

  • ASP

    서버에서 동적으로 데이터를 처리하고 웹 페이지에 출력했습니다.

라이브러리

  • Swiper

    콘텐츠 슬라이더 구현에 사용했습니다.

  • Bootstrap

    빠른 레이아웃 구성과 반응형 웹 구현에 사용했습니다.

  • AOS

    스크롤 시 애니메이션 효과에 사용했습니다.

  • Slick Slider

    모바일 터치 슬라이드 구현에 사용했습니다.

디자인

  • Photoshop

    웹 이미지 편집 및 배너 제작에 사용했습니다.

  • Illustrator

    아이콘, 로고 등 벡터 이미지 제작에 사용했습니다.

  • Adobe XD

    와이어프레임 및 UI 시안 제작에 사용했습니다.

협업

  • GitHub

    커뮤니케이션 및 프로젝트 협업에 사용했습니다.

  • Notion

    정리 및 업무 내용 작성, 공부에 사용했습니다.

  • File Zilla

    웹 호스팅서버에 파일 전송 및 수정 작업에 사용했습니다.
